Game Controls:

Player 1 Controls:

A: Move Left

D: Move Right

W: Jump

Space: Kick

Player 2 Controls:

Left Arrow (<): Move Left

Right Arrow (>): Move Right

Up Arrow: Jump

Down Arrow: Kick
